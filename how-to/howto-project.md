# How-to: Project

## Descripción
Este documento contiene las reglas y guías para crear y gestionar proyectos.

## Reglas

### Estado del Proyecto
- El proyecto debe crearse siempre en estado **"Shaping"**

### Equipo de Linear
- Debes preguntar al usuario en qué equipo (de los ya existentes en Linear) quiere crear el proyecto
- Lista los equipos disponibles en Linear para que el usuario elija
- Pregunta quién debería ser el lider del proyecto (una un list users cuando te den el nombre)
